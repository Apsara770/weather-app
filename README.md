
# 🌤 Weather App using HTML, CSS & JavaScript

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript** with real-time weather data from the OpenWeatherMap API.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡 Real-time temperature display  
- ☁️ Weather condition (Cloudy, Rain, Clear, etc.)  
- 🎯 API integration (OpenWeatherMap)  
- 📱 Simple and responsive UI  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- OpenWeatherMap API
